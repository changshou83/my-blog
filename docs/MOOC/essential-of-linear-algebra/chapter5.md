# 行列式

- 二维空间下行列式的意义
	- 变换之后的面积是变换之前的多少倍
		- 三维的话就是体积
	- 正负表示空间的定向(就像一张纸的正反面，可以通过基向量的位置关系来区分)
		- 三维空间下就需要左/右手坐标系，如果变换之后不能用同一只手表示基向量，那么就是负值
- 计算
	- 二维
		- 矩阵
			- `[a b]`
			- `[c d]`
		- 行列式：`det(matrix) = ad - bc`
