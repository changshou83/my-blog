方法重载

  • ts 中会发生方法重载：同名函数根据不同参数列表执行不同操作

WARNING

最下面的要兼容上面的，不然会报错：此重载签名与其实现签名不兼容

 
 
 



function getInfo(name:string):string;
function getInfo(age:number):number;
function getInfo(name:any, age?:any):any {
  return age ? `${name} --- ${age}` : `${name} --- 年龄保密`
}
